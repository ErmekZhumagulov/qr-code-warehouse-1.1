Инструкция по использованию

- все репорты и данные сохраняются исключительно в этой папке
- файл main.py является скриптом программы, рекомендуется не редактировать
- файл ui.ui является файлом интерфейса программы, и редактируется в программе Qt Designer
- файл logs.csv является логами журнала в данный момент
- папка archive - папка, в котором хранятся архивные данные репорта журнала
- файлы:
	Пароль.exe
	Список оборудования.exe
	Список получателей.exe
	Список сотрудников.exe
		- Это файлы редактируемые и несут информацию для записей в журнал.
		- Можно проводить редактирования посредством удаления, добавления или изменения данных.
		- Добавление новых данных проводить с новой строки.
